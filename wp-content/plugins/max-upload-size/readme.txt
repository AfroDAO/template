Max Upload Size plugin allows you to change upload_max_filesize for wordpress through plugin.

== Description ==

You can maximize the upload_max_filesize for your wordpress website through this plugin. 

Major features of this plugin include:

* Maximize upload_max_filesize

== Installation ==

* Upload the Max Upload Size plugin to the `/wp-content/plugins/` directory
* Activate the Max Upload Size plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to increase upload_max_filesize ? =
* Go to Wordpress menu - Max Upload Size
* Only enter the numeric value in bytes
* 1024 bytes = 1KB
* 1048576 bytes = 1MB
* 1048576000 bytes = 1000MB
